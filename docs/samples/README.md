# Sample Data Files

This directory contains sample CSV data files used for testing and reference.

## Files

- `exported_csv_sample.csv` - Sample export of SAT questions
- `latest_export.csv` - Latest question export sample
- `test_export.csv` - Test data export sample

## Purpose

These files are:
- Used as reference for data format
- Sample data for testing import/export functionality
- **Not committed to git** (see .gitignore)

## Note

⚠️ CSV files in this directory are gitignored to prevent repository bloat.
If you need sample data, you can regenerate it using the export functionality or contact the team.
