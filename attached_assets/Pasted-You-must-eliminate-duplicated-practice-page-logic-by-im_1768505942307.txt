## Step 1: Create canonical hook

client/src/hooks/useCanonicalPractice.ts

- measure elapsedMs using a ref questionStartMs
- call GET /api/practice/next?section=<section>

  { questionId, selectedAnswer, freeResponseAnswer, elapsedMs, skipped }
- accept optional server stats from either endpoint and adopt it if present; otherwise fallback to current client increment logic.
- expose handleMissingMcChoices that triggers submitAnswer({skipped:true}) for MC questions.

Implementation must match the exact content provided in the spec (do not invent types).
